# 灯效使用说明 — 12种灯光效果详解

---

## 1. STATIC — 静态颜色

| 项目 | 说明 |
|------|------|
| **中文名称** | 静态颜色 |
| **肉眼看到的效果** | 所有灯带显示同一种固定颜色，不会变化 |
| **是否需要视频** | 不需要 |
| **是否需要音频** | 不需要 |
| **视频控制什么** | 无 |
| **音频控制什么** | 无 |
| **源码文件** | `light_engine/effects/static.py` |
| **核心参数** | `color` — RGB颜色值（默认蓝色 [0.2, 0.4, 0.8]） |
| **启动命令** | `.python\python.exe -m light_engine demo --effect static` |
| **无视频/无音频时** | 不受影响，始终显示固定颜色 |
| **适合场景** | 基础照明、测试灯具安装、调试 |

## 6. AUDIO_PULSE — 音乐呼吸

| 项目 | 说明 |
|------|------|
| **中文名称** | 音乐呼吸灯 |
| **肉眼看到的效果** | 灯光亮度跟随音乐整体能量起伏 |
| **是否需要视频** | 不需要 |
| **是否需要音频** | **需要** |
| **视频控制什么** | 无 |
| **音频控制什么** | RMS总体能量→亮度；有attack/release包络平滑 |
| **源码文件** | `light_engine/effects/audio_pulse.py` |
| **核心参数** | `attack` — 攻击速度, `release` — 释放速度, `color` — 基础颜色 |
| **启动命令** | `.python\python.exe -m light_engine run --audio 音乐.wav --effect audio_pulse` |
| **无音频时** | 保持低亮度，不会闪烁 |
| **适合场景** | 音乐播放时使用 |

## 8. SPECTRUM — 频谱映射

| 项目 | 说明 |
|------|------|
| **中文名称** | 频谱分区灯 |
| **肉眼看到的效果** | 不同区域分别显示低、中、高频的能量 |
| **是否需要视频** | 不需要 |
| **是否需要音频** | **需要** |
| **视频控制什么** | 无 |
| **音频控制什么** | 低频→天花板区域(红色)，中频→墙面区域(绿色)，高频→前后区域(蓝色) |
| **源码文件** | `light_engine/effects/spectrum.py` |
| **核心参数** | `bass_zones` / `mid_zones` / `treble_zones` — 各频段对应区域 |
| **启动命令** | `.python\python.exe -m light_engine run --audio 音乐.wav --effect spectrum` |
| **无音频时** | 全暗 |
| **适合场景** | 音乐可视化、音频调试 |

## 9. VIDEO_AMBIENT — 视频环境光

| 项目 | 说明 |
|------|------|
| **中文名称** | 视频环境光 |
| **肉眼看到的效果** | 灯带颜色跟随视频画面各区域的实际颜色 |
| **是否需要视频** | **需要** |
| **是否需要音频** | 不需要 |
| **视频控制什么** | 分区颜色→各灯带颜色；有平滑处理避免闪烁 |
| **音频控制什么** | 无 |
| **源码文件** | `light_engine/effects/video_ambient.py` |
| **核心参数** | `smoothing` — 平滑程度(越大越慢) |
| **启动命令** | `.python\python.exe -m light_engine run --video 视频.mp4 --effect video_ambient` |
| **无视频时** | 显示极暗的默认色 |
| **适合场景** | 观影环境光、视频氛围增强 |

## 10. VIDEO_AUDIO_FUSION — 视频音乐融合（核心模式）

| 项目 | 说明 |
|------|------|
| **中文名称** | 视频音乐融合灯 |
| **肉眼看到的效果** | 视频决定基础颜色，音乐决定亮度、脉冲、闪烁 |
| **是否需要视频** | **需要** |
| **是否需要音频** | **需要** |
| **视频控制什么** | 基础色相、各区域颜色 |
| **音频控制什么** | RMS→亮度，低频→中心脉冲，中频→饱和度，高频→像素微闪(有上限)，节拍→短闪光 |
| **源码文件** | `light_engine/effects/video_audio_fusion.py` |
| **核心参数** | `video_weight`(0.65), `audio_weight`(0.35), `bass_boost`(1.5), `treble_limit`(0.4) |
| **启动命令** | `.python\python.exe -m light_engine run --video 视频.mp4 --audio 音频.wav --effect video_audio_fusion` |
| **无视频时** | 退化为纯音频灯效 |
| **无音频时** | 退化为纯视频环境灯 |
| **两者都无** | 自动切换到CALM模式 |
| **适合场景** | **主要演示模式**，MV、演唱会视频 |
