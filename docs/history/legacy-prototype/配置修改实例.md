# 配置修改实例

> 修改配置文件后，保存文件，重新双击对应的 `.bat` 文件即可看到变化。

---

## 1. 将全局亮度从85%改为30%

**修改文件**: `config\system.yaml`

**修改前**:
```yaml
  smoothing:
    max_brightness: 0.85
```

**修改后**:
```yaml
  smoothing:
    max_brightness: 0.3
```

**运行**: 双击 `01_启动演示.bat`
**看到的变化**: 所有灯光明显变暗，最高亮度只有原来的35%。

---

## 2. 将输出FPS从30改为20

**修改文件**: `config\system.yaml`

**修改前**:
```yaml
system:
  output_fps: 30.0
```

**修改后**:
```yaml
system:
  output_fps: 20.0
```

**运行**: 双击 `01_启动演示.bat`
**看到的变化**: 灯光更新速度变慢，每帧间隔从33ms变为50ms。在模拟器中FPS显示约20。

---

## 3. 将跑马灯速度减半

**修改文件**: `config\effects.yaml`

**修改前**:
```yaml
  chase:
    speed: 2.0
```

**修改后**:
```yaml
  chase:
    speed: 1.0
```

**运行**: `.python\python.exe -m light_engine demo --effect chase`
**看到的变化**: 光点移动速度减半。

---

## 4. 将灯带像素数从144改为120

**修改文件**: `config\layout.yaml`

**修改前**:
```yaml
  strips:
    - id: "ceiling_left"
      type: "digital"
      pixel_count: 144
```

**修改后**:
```yaml
  strips:
    - id: "ceiling_left"
      type: "digital"
      pixel_count: 120
```

**运行**: 双击 `02_启动模拟器.bat`
**看到的变化**: 天花板左灯带缩短，从144像素变为120像素。

---

## 5. 增加一个新区域

**修改文件**: `config\layout.yaml`

在 `zones:` 和 `strips:` 列表中分别增加一个区域：

```yaml
  zones:
    - id: "floor"
      type: "rgbw"
      label: "Floor Panel"

  strips:
    - id: "floor"
      type: "digital"
      pixel_count: 100
      direction: "forward"
      video_zone: "bottom"
```

**运行**: 双击 `01_启动演示.bat`
**看到的变化**: 模拟器多显示一行地板灯带。

---

## 6. 将灯带方向反转

**修改文件**: `config\layout.yaml`

**修改前**:
```yaml
    - id: "ceiling_right"
      direction: "forward"
```

**修改后**:
```yaml
    - id: "ceiling_right"
      direction: "reverse"
```

**运行**: `.python\python.exe -m light_engine demo --effect chase`
**看到的变化**: 跑马灯在天花板右灯带上反向移动。

---

## 7. 将视频权重改为70%

**修改文件**: `config\effects.yaml`

**修改前**:
```yaml
  video_audio_fusion:
    video_weight: 0.65
    audio_weight: 0.35
```

**修改后**:
```yaml
  video_audio_fusion:
    video_weight: 0.70
    audio_weight: 0.30
```

**运行**: `.python\python.exe -m light_engine run --video xxx.mp4 --audio xxx.wav --effect video_audio_fusion`
**看到的变化**: 视频颜色对灯光的影响更强，音频能量影响减弱。

---

## 8. 将音频权重改为30%

同上，把 `audio_weight` 从 0.35 改为 0.30。

---

## 9. 修改串口号

**修改文件**: `config\outputs.yaml`

**修改前**:
```yaml
  serial:
    port: "COM3"
```

**修改后**:
```yaml
  serial:
    port: "COM5"
```

**注意**: 当前串口仅软件模拟。需要连接STM32硬件后才能真实通信。

## 11. 恢复默认配置

如果改乱了，可以恢复默认配置：

1. 删除 `config` 文件夹
2. 从备份或Git历史恢复原始 `config` 文件夹

或者只恢复单个文件：
- 每个配置文件都有默认值写在源码中
- 删掉某个配置文件后，程序会使用默认值

**确认恢复成功**: 运行 `03_运行测试.bat`，如果162个测试全部通过，说明配置正常。
