# 程序调用关系图

## 当前DEMO实际经过的模块

```mermaid
flowchart TD
    CLI[CLI入口: cli/__init__.py] --> Config[配置加载: config/__init__.py]
    CLI --> Engine[Engine: engine/__init__.py]
    Engine --> DataSource[合成数据源: data/generators.py]
    Engine --> Effect[当前灯效: effects/demo.py]
    Effect --> Static[static.py]
    Effect --> Breath[breath.py]
    Effect --> ColorWave[color_wave.py]
    Effect --> Chase[chase.py]
    Effect --> Comet[comet.py]
    Effect --> AudioPulse[audio_pulse.py]
    Effect --> Spectrum[spectrum.py]
    Effect --> Calm[calm.py]
    Effect --> VideoFusion[video_audio_fusion.py]
    Effect --> PixelFrame[PixelFrame: models.py]
    Engine --> Outputs[输出路由: outputs/__init__.py]
    Outputs --> JsonOut[JsonOutput: outputs/json_output.py]
    Outputs --> SimOut[SimulatorOutput: outputs/simulator_output.py]
    PixelFrame --> JsonOut
    PixelFrame --> SimOut
```

## 视频模式实际经过的模块

```mermaid
flowchart TD
    CLI --> Config
    CLI --> Engine
    Engine --> VideoReader[VideoReader: media/__init__.py]
    Engine --> VideoAnalyzer[VideoAnalyzer: analysis/video.py]
    VideoReader --> Frame[BGR帧数组]
    Frame --> VideoAnalyzer
    VideoAnalyzer --> VideoFeatures[VideoFeatures: models.py]
    VideoFeatures --> EffectContext[EffectContext: models.py]
    EffectContext --> Effect
    Effect --> PixelFrame
    Engine --> Outputs
    Outputs --> JsonOut
    Outputs --> SimOut
```

## 音频模式实际经过的模块

```mermaid
flowchart TD
    CLI --> Config
    CLI --> Engine
    Engine --> AudioReader[AudioReader: media/__init__.py]
    Engine --> AudioAnalyzer[AudioAnalyzer: analysis/audio.py]
    AudioReader --> Samples[浮点采样数组]
    Samples --> AudioAnalyzer
    AudioAnalyzer --> AudioFeatures[AudioFeatures: models.py]
    AudioFeatures --> EffectContext
    EffectContext --> Effect
    Effect --> PixelFrame
    Engine --> Outputs
```

## 目前不会在默认DEMO中运行的模块

| 模块 | 原因 |
|------|------|
| `outputs/udp_output.py` (UDP) | 需ESP32-S3硬件 |
| `outputs/serial_output.py` (串口) | 需STM32硬件 |
| `clock.py` (时钟系统) | 当前未集成到Engine |
| `pipeline.py` (队列架构) | 当前未集成 |

## 只有真实硬件接入后才有作用的模块

| 模块 | 需要什么硬件 |
|------|-------------|
| `UdpOutput` | ESP32-S3 + Wi-Fi + WS2812B数字灯带 |
| `SerialOutput` | STM32F103C8T6 + RS-485 + RGBW COB灯带 |
